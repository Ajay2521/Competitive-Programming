## Multiplication without using multiplication operator

For given m, n values return the multiplication of m * n with out using a multiplication operator

**Example 1:**

```

Input: m = 2, n = 3

Output: 6

```

**Example 2:**

```

Input: m = 3, n = 9

Output: 9

```
